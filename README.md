# fos

A folder archive tool for Linux or Mac

[![Build Status](https://travis-ci.org/scottyg/fos.svg?branch=master)](https://travis-ci.org/scottyg/fos)

## Installation

    $ gem install fos

## Usage

    $ fos
or
    $ fos -h
